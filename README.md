# SUGAM - Smart University Governance and Approval Management

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `streamlit run main.py`

## Project Structure
- `main.py` - Main application entry point
- `core/` - Core application logic and database operations
- `models/` - AI models and RAG system
- `modules/` - Streamlit page modules
- `institution/` - Institution portal modules
- `utils/` - Utility functions
- `data/` - Database and data files
- `assets/` - Static assets

## Features
- Institutional Performance Analytics
- AI-Powered Document Analysis
- Institutional Intelligence Hub
- Data Management Tools
- Institution Portal with Document Upload
- API Documentation
- PDF Report Generation
